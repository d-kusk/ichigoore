# Ichigoore
> A boilerplate of gulp for FICC KYOTO Development Team

## Features
- Sass(.scss)
- CoffeeScript
- Imagemin
- Concat vendors
- BrowserSync

## Install
```
$ git clone git@github.com:p1ch-jp/ichigoore.git
```

[More...](https://www.gitbook.com/book/d-kusk/gulp-setting/details)

## Usage
```
$ npm install
$ gulp
```
